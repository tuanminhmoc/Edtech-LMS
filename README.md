# EdTech LMS Pro

## Chạy local

```bash
python -m http.server 8080
```

Mở `http://localhost:8080`.

## Deploy GitHub Pages

1. Đẩy toàn bộ thư mục lên repository.
2. Vào **Settings → Pages**.
3. Chọn **GitHub Actions**.
4. Push lên nhánh `main` hoặc `master`.

## Kiểm tra code

```bash
npm install
npm run check
npm test
```

## Dữ liệu

Dữ liệu nằm trên trình duyệt hiện tại. Dùng nút **Dữ liệu** để sao lưu trước khi đổi máy hoặc xóa dữ liệu trình duyệt.
