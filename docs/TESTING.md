# Kiểm thử

```bash
npm run check
npm run fixtures
npx playwright install chromium
npm run visual:update
npm run visual
```

Visual regression chụp:

- 1440×900
- 1366×768
- 768×1024
- 360×800
- 390×844

Màn hình: Dashboard, Cấu hình, Trắc nghiệm, Kết quả, Flashcard trước/sau, Thư viện, Soạn đề, Lịch sử và Settings.
